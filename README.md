## 🚀 About This Project

This is my personal portfolio website built with React.js and Vite, featuring a premium dark theme with glassmorphism effects. The portfolio showcases my skills, projects, and experience as a Java Full Stack Developer with expertise in Spring Boot, React.js, and AI integration.

## ✨ Features

- **Premium Dark Theme** - Glassmorphism UI with cyan and purple gradient accents
- **Smooth Animations** - Fade-in effects, animated progress bars, and counter animations
- **Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- **SEO Optimized** - Meta tags and semantic HTML for better search visibility
- **Fast Performance** - Built with Vite for lightning-fast load times
- **10 Sections** - Hero, About, Skills, Metrics, Projects, Experience, Why Hire Me, Achievements, Contact, Footer

## 📦 Installation & Setup

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Steps to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/abishek-palanivel/My-portfolio.git
   cd My-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Add your profile photo (Optional)**
   - Place your photo as `public/profile.jpg`
   - Recommended size: 500x500px
   - Or update the image URL in `src/components/Hero.jsx`

4. **Start development server**
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:5173`

5. **Build for production**
   ```bash
   npm run build
   ```
   Production files will be in the `dist` folder

6. **Preview production build**
   ```bash
   npm run preview
   ```
