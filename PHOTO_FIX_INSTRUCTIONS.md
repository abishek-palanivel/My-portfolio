## To Make Photo Appear:

### Option 1: Upload to GitHub (Recommended)
1. Save your photo as `profile.jpg`
2. Go to your GitHub repository
3. Navigate to the `public` folder
4. Drag and drop `profile.jpg` into the folder
5. Commit the changes
6. Wait 2-3 minutes for Netlify to rebuild

### Option 2: Test Locally First
1. Save your photo as `profile.jpg`
2. Put it in the `public` folder on your computer
3. Run: `npm run dev`
4. Check if it appears at http://localhost:5173
5. If it works, push to GitHub
